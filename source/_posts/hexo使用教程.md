# 开始

## 进入 d:\blog 文件夹

hexo 必须在这个目录下才可运行

## hexo s

启动 hexo

## hexo n "博客名"

通过 hexo 生成.md 文档，修改 tags 为 category 进行分类

## 将文件部署到 github 公共域名上

hexo clean 进行清理缓存等，再通过 hexo d 进行部署 a

## 修改名字

在 d:\blog 下找到\_config.yml，修改 author 名字，即可换作者名称。
